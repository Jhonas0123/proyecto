# Aplicación Educativa de Matemáticas - Frontend

Este directorio contiene la parte frontend de la Aplicación Educativa de Matemáticas, diseñada para ayudar a estudiantes de primaria a aprender matemáticas básicas mediante ejercicios interactivos y contenido atractivo.

## Estructura del Proyecto

- `index.html`: Archivo HTML principal que sirve como punto de entrada de la aplicación.
- `css/styles.css`: Contiene los estilos de la aplicación, asegurando un diseño moderno y adaptable.
- `js/app.js`: Archivo JavaScript que gestiona las interacciones del usuario y actualizaciones dinámicas.
- `assets/`: Directorio para imágenes, fuentes y otros recursos.

## Primeros Pasos

Para comenzar con el frontend de la Aplicación Educativa de Matemáticas, sigue estos pasos:

1. **Clona el repositorio**: 
   ```
   git clone <url-del-repositorio>
   cd math-education-app/frontend
   ```

2. **Abre el archivo `index.html`**: Puedes abrir este archivo en cualquier navegador web para ver la aplicación.

3. **Personaliza los estilos**: Modifica el archivo `css/styles.css` para cambiar la apariencia de la aplicación según tus necesidades.

4. **Agrega interactividad**: Utiliza el archivo `js/app.js` para implementar nuevas funciones o modificar las existentes.

## Características

- Ejercicios matemáticos interactivos para estudiantes de primaria.
- Retroalimentación inmediata sobre las respuestas.
- Diseño responsivo para accesibilidad en distintos dispositivos.
